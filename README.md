#Interactive Bitcoin Insights (2014–2026) — A Power BI Dashboard for Trends & Market Analysis

This repository contains an interactive Power BI dashboard built on a Bitcoin dataset spanning 2014–2026. The project showcases key financial and market insights through dynamic visualizations and KPI cards, designed for both analysis and presentation purposes.

Features
Price Trends → Daily, weekly, and monthly closing price analysis

Trading Volume Insights → Growth %, spikes, and liquidity tracking

Market Capitalization → Value trends and growth comparisons

Volatility Metrics → High–Low spread %, drawdowns, and risk indicators

Comparative Analysis → Bitcoin vs USD/INR, Gold, and other assets

Purpose
Demonstrates data analytics and visualization skills using Power BI

Highlights cryptocurrency market behavior over a 12-year period

Serves as a portfolio project for resume and professional showcase
